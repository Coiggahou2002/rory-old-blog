Java容器总结

Coiggahou
2022-08-22
Java

UNDER CONSTRUCTION …

Java 中的容器主要分为 Collection 和 Map

Collection

List

实现：

  • ArrayList
  • LinkedList

Set

  • HashSet
  • LinkedHashSet
  • TreeSet

Queue

Deque

Deque 是一个继承自 Queue 的接口，可选实现有 LinkedList 与 ArrayDeque

一般选择 ArrayDeque，比 LinkedList 实现要快

This class is likely to be faster than Stack when used as a stack, and faster than LinkedList when used as a queue.

——JDK 1.8 文档注释

作为 Deque 的实现类，LinkedList 比 ArrayDeque 慢的原因主要是：

  • 队列经常需要在头尾增加节点，链式结构的头尾增删需要唤醒 JVM 和 OS 进行内存分配
  • 链式结构删除节点的时候，背后 GC 需要进行内存回收，增大开销
  • 链式结构本身的 cache-friendliness 不如数组的连续内存结构

ArrayDeque

  • 线程不安全
  • 自动扩容
  • 作为栈使用时比 Stack 快
  • 作为队列使用时比 LinkedList 快

Map

  • Hashtable
  • HashMap
  • LinkedHashMap
  • TreeMap
