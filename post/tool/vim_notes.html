Vim速成笔记

Coiggahou
2022-07-27
Linux, Vim

All Modes:

  • Normal Mode

  • Command Line Mode

  • Insert Mode (for insert and edit)

  • Visual Mode (for selecting chunks of text)

u to undo

^r to redo

Moving around the cursor

Move by character

j to go down

k to go up

h to go left

l to go right

gg to go to the top of the file

G to go to the bottom of the file

^D to go down faster

^U to go up faster

Move by word

w to go right by a step of a word, b to go back reverse

0 moves the cursor to the beginning of a line

^ moves the cursor to the first char of a line

$ moves the cursor to the end of a line

Editing

i to insert mode

a to insert mode, but will move the cursor to append sth after the current char

o to insert a new line BELOW the current line and switch to insert mode

O to insert a new line ABOVE the current line and switch to insert mode

x to delete a single char (at normal mode)

dd to delete the current line

r[char] to replace the current char with another

Copy and Paste

y for copy (yank means copy)

yy to copy current line

p to paste

: to command line mode

Selecting chunks

v to enter regular visual mode, then move the cursor to select chunks

V to enter visual line mode (every movement selects a whole line)

:q to quit

:q! to quit discarding any changes

:qa to quit all

:w to save

:wq to save and quit

Repeating operations

just press the times you want to repeat before the operation command

Modifier

a most common modifying requirement is that we need to change the thing in a pair of bracket (or any other pair of wrappers)

for example, ci[ to change the inside of a pair of []

di[ to delete the inside of []

da[ to delete the inside and wrapper of [xxx]

i means inside, a means around

Search and replace

/ to search, enter the target then press Enter, then n or N to switch between matched results

R to replace the current char with more than one chars

:%s/<target>/<substitution>/g to do global substitution (to change target to substitution)

Retrieve

r to retrieve sth

for example, we can retrieve the output of an external command like !ls

Others

:! to run external commands

:sp to separate the same file to 2 windows

:tabnew to open a new tab

gt to switch betwenn tabs

:help to see help
