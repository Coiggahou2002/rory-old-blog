JavaScript | for...of 与 for...in 的区别

Coiggahou
2022-03-06
前端开发
JavaScript, 前端开发

要搞明白两者的区别，我们要先分别搞清楚他们各自能做什么事情

for…of

for…of 用于遍历可迭代对象

常见的可迭代对象有: Array, String, array-like Objects, TypedArray, Map, Set

MDN 原话：

The for…of statement creates a loop iterating over iterable objects, including: built-in String, Array, array-like objects (e.g., arguments or NodeList), TypedArray, Map, Set, and user-defined iterables. It invokes a custom iteration hook with statements to be executed for the value of each distinct property of the object.

for…in

for…in 用于遍历一个对象中的所有可枚举属性的 key（包括其原型链上的可枚举属性）

从这张图中可以验证：

  • for-in 连对象的原型链上的可枚举属性一起遍历
  • 只有可枚举(enumerable)的属性能被遍历

但是，如果我们只想遍历一个对象的所有可枚举属性 key，就需要绕过原型链的可枚举属性

总结

在了解完 for-in 和 for-of 分别能做什么之后，我们来总结一下这两个语法点

  • for-in 和 for-of 都是在做一个遍历/迭代行为，只是遍历的东西不一样
  • for-of 用于遍历任何可迭代对象(Iterable Objects)，遍历出来的东西由这个对象内部的迭代器决定（它定义了迭代行为能够迭代出什么样的数据，我们用 for-of 就会迭代出什么样的数据）
  • for-in 用于遍历一个对象及其原型链上的所有可枚举属性(enumerable properties)的 key
