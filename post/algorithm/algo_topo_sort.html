有向无环图の拓扑排序

Coiggahou
2021-10-16
算法
图, 数据结构, 算法

概述

拓扑排序只适用于 DAG（有向无环图）

步骤：

  1. 将所有入度为 0 的顶点入队
  2. 队列中逐个顶点出队，删掉与该顶点关联的所有边，同时维护对应点的入度（删一条边，入度减 1）
  3. 如果删边过程中发现对面的点入度为 0，将它入队
  4. 比较图的顶点数与拓扑序列的长度，相等则可拓扑，否则该图不存在拓扑序列

拓扑排序常用框架是 BFS，时间复杂度为 $O(n+m)$，可用于有向连通图判环.
